== README

* Ruby version 2.2.1

Instruction:

Load rails server from the rails_todo_html directory. 
In browser go to http://localhost:3000. 
Todo list will be displayed from the tasks table in the database. 
Clicking on a task will display the task on a separate page. 
To add another task to the list, type a task into the New Task box at the bottom and press save. 
List will be reloaded with the the task at the bottom.
